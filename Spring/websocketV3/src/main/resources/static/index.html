Content-Type: text/event-stream
브라우저가 이 헤더를 보면 스트리밍 모드로 전환합니다.
서버는 연결을 유지한 채 언제든 이벤트를 보낼 수 있습니다.

Long Polling vs SSE 핵심 차이:
Long Polling: 응답 → 연결 종료 → 재연결
SSE: 응답 → 연결 유지 → 다음 응답 → 연결 유지

브라우저 자동 재연결:
EventSource API는 연결이 끊기면
자동으로 재연결하고 Last-Event-ID를
헤더로 전송해 놓친 메시지를 복구합니다.
